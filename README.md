Resume
======

> My full resume in the JSON-based jsonresume.org format.

[![Build Status](https://travis-ci.org/iGitScor/resume.svg?branch=master)](https://travis-ci.org/iGitScor/resume)

## Getting started

```
npm install
npm run export
```

### Other language

```
npm run export -- --lang=fr
```
